# 1045. Customers Who Bought All Products

[題目出處](https://leetcode.com/problems/customers-who-bought-all-products/description/?envType=study-plan-v2&envId=top-sql-50)

## 題目
Table: `Customer`

| Column Name | Type |
|-------------|------|
| customer_id | int  |
| product_key | int  |

This table may contain duplicates rows. \
customer_id is not NULL.\
product_key is a foreign key (reference column) to Product table.

Table: `Product`
| Column Name | Type |
|-------------|------|
| product_key | int  |

product_key is the primary key (column with unique values) for this table.

請寫一個解法，找出 `Customer`表中，購買了 `Product`表中所有產品的顧客 ID `customer_id`。\
回傳所有符合條件的 `customer_id`\
結果順序不限

## 解題思路
先理解目標是找出「購買了所有商品的顧客」，因此要先知道 Product 表中共有多少種不同商品，再對 Customer 表依 customer_id 進行分組，計算出每位顧客購買的不同類 product_key 數量（需使用 DISTINCT 避免重複購買同一商品影響結果）\
最後將每位顧客購買的商品種類數與 `Product` 表的總商品數進行比較，若相等則代表該顧客買齊所有商品

## SQL 寫法

```sql
select customer_id from Customer
group by  customer_id
having count(distinct product_key) = (
    select count(*) from Product
)
```
