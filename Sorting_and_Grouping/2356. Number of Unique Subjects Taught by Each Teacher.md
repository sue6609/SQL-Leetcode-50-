# 2356. Number of Unique Subjects Taught by Each Teacher


[題目出處](https://leetcode.com/problems/number-of-unique-subjects-taught-by-each-teacher/description/?envType=study-plan-v2&envId=top-sql-50)




## 題目
Table: `Teacher`

| Column Name | Type |
| ----------- | ---- |
| teacher_id  | int  |
| subject_id  | int  |
| dept_id     | int  |


在這個資料表中，**(subject_id, dept_id)** 是主鍵（Primary Key）。


每一筆資料表示某位老師在某個部門教授某個科目。
請找出每位老師教授了多少個**不同的科目**。
結果順序不限。

 

## 解題思路
題目要統計每位老師教授了多少個不同的科目。
由於同一位老師可能在不同部門教授同一門課，因此需要使用 COUNT(DISTINCT subject_id) 來計算不重複的科目數量，再依照 teacher_id 分組統計即可。


## SQL 寫法

```sql
SELECT teacher_id,
       count(DISTINCT subject_id) AS cnt  
FROM Teacher 
GROUP BY teacher_id;
```
