# 1757. Recyclable and Low Fat Products

[題目出處](https://leetcode.com/problems/recyclable-and-low-fat-products/description/?envType=study-plan-v2&envId=top-sql-50)

## 題目

表格：`Products`

| 欄位名稱 | 型別 |
|----------|------|
| product_id | int |
| low_fats | ENUM('Y', 'N') |
| recyclable | ENUM('Y', 'N') |

找出同時符合以下條件的產品：

- `low_fats = 'Y'`（低脂）
- `recyclable = 'Y'`（可回收）

回傳這些產品的 `product_id`。


## 解題思路

這題是 **SELECT + WHERE** 練習。

條件要求：

1. 產品必須是低脂 (`low_fats = 'Y'`)
2. 產品必須可回收 (`recyclable = 'Y'`)

由於兩個條件必須同時成立，因此使用 `AND`。



## SQL 寫法

```sql
SELECT product_id
FROM Products
WHERE low_fats = 'Y'
  AND recyclable = 'Y';
```
